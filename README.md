# PyStreamify - Streamlit

## Setup Python Environment

```bash
py -m venv venv
venv\Scripts\activate
python -m pip install --upgrade pip
```

### Check if Python is installed

```bash
python --version
```

## How to install Streamlit

```bash
pip install streamlit
streamlit hello
```
